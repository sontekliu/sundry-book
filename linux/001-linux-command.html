01-Linux常用命令总结

1. 安装 rz、sz 命令

在使用 SecureCRT 工具时，上传下载文件一般使用 rz 和 sz。安装如下：

yum install -y lrzsz

打开SecureCRT软件 -> Options -> session options -> X/Y/Zmodem 下可以设置上传和下载的目录
