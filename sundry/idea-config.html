IDEA 配置

IDEA 安装完成之后的配置

  1. 配置字体 Settings -> Editor -> Font
  2. 文件编码 Settings -> Editor -> File Encodings
  3. 修改换行，即 CRLF -> LF Settings -> Editor -> Code Style
  4. 去除拼写检查 Settings -> Editor -> Inspections -> 输入 typo 将打钩去掉即可
  5. 关闭启动时候打开最近项目 Appearance & Behavior -> System Settings 去掉打钩(Reopen last project on startup)
  6. 配置 Maven Settings -> Build, Execution, Deployment -> Build Tools -> Maven
  7. 开启自动编译 Settings -> Build, Execution, Deployment -> Compiler -> Build project automatically
