# yarn

### 1. yarn install

```shell
$ brew install yarn
```

### 2. 安装依赖

```shell
$ yarn     // 等于 npm install 
```

### 3. 构建/启动应用

```shell
$ yarn build
$ yarn start
```

### 4. 安装、卸载、升级模块

```shell
$ yarn add react
$ yarn global add create-react-app
$ yarn remove react
$ yarn add react --dev
$ yarn upgrade
```
