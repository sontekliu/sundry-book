01-yarn

yarn

1. yarn install

$ brew install yarn

2. 安装依赖

$ yarn     // 等于 npm install

3. 构建/启动应用

$ yarn build
$ yarn start

4. 安装、卸载、升级模块

$ yarn add react
$ yarn remove react
$ yarn add react --dev
$ yarn upgrade
